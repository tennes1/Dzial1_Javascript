1) Kiedy powstał JavaScript (podaj datę).

JavaScript powstał w 1995 roku.

2) Dlaczego jest to język skryptowy

JavaScript jest językiem skryptowym, ponieważ jest interpretowany i wykonywany bezpośrednio przez przeglądarkę, bez potrzeby wcześniejszej kompilacji.

3) Gdzie jest wykonywany JS (klient lub serwer)

JavaScript jest wykonywany po stronie klienta, czyli w przeglądarce użytkownika.

4) Dlaczego jest JS to bezpieczne narzędzie

JavaScript jest bezpieczne, ponieważ działa w tzw. "piaskownicy" (sandbox), która ogranicza dostęp do systemu operacyjnego i chroni użytkownika przed nieautoryzowanym dostępem.

5) Czy jest to język obiektowy, jeśli tak to dlaczego.

Tak, JavaScript jest językiem obiektowym, ponieważ opiera się na obiektach i prototypach, pozwalając na tworzenie i manipulowanie obiektami.

6) Wymień zalety JS:

Zaleta1: Jest wykonywany po stronie klienta, co zmniejsza obciążenie serwera.

Zaleta2: Pozwala na dynamiczne zmiany zawartości strony bez przeładowania.

Zaleta3: Jest szeroko wspierany przez wszystkie nowoczesne przeglądarki.

7) Framework to:

Framework to gotowy zestaw narzędzi i bibliotek, który ułatwia tworzenie aplikacji poprzez dostarczenie struktury i gotowych rozwiązań.

8) Do aplikacji mobilnych można użyć Framework

Do aplikacji mobilnych można użyć frameworków takich jak React Native, Ionic czy Flutter.

9) Aplikacje responsywne to:

Aplikacje responsywne to takie, które automatycznie dostosowują swój wygląd i układ do rozmiaru ekranu urządzenia, na którym są wyświetlane.

10) Przepisz linię kodu pod nią wytłumaczenie jak działa ten fragment kodu

<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

Jest to przycisk, który po kliknięciu wywołuje funkcję WinOpen z pustym argumentem, co prawdopodobnie otwiera nowe okno lub stronę.

11) Przepisz linię kodu pod nią wytłumaczenie jak działa ten fragment kodu

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Ta instrukcja otwiera nowe okno przeglądarki z zawartością pliku "obraz.html", o nazwie "okienko" i z ustawionymi opcjami:

  • toolbar=no - ukrywa pasek narzędzi przeglądarki
  • directories=no - ukrywa przyciski katalogów
  • menubar=no - ukrywa menu przeglądarki
  • height=280 - ustawia wysokość okna na 280 pikseli
  • width=160 - ustawia szerokość okna na 160 pikseli
  • top=200 - pozycjonuje okno 200 pikseli od góry ekranu
  • left=200 - pozycjonuje okno 200 pikseli od lewej strony ekranu

12) Opisz następującą instrukcję window.close()

Instrukcja window.close() zamyka aktualnie otwarte okno przeglądarki, w którym została wywołana.
